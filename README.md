# Typing Speed Tester

## Overview

This is a simple terminal-based Typing Speed Tester built using Python. The program displays a random paragraph and measures the total time taken by the user to type it. It is designed as a beginner-friendly project to practice Python fundamentals and understand how to work with time measurement and random text selection.

## Features

* Displays a random paragraph.
* Starts the timer when the user begins the test.
* Stops the timer after the user submits the typed text.
* Calculates and displays the total time taken.
* Runs entirely in the terminal.
* Uses only Python's built-in libraries.

## Technologies Used

* Python 3
* `time` module
* `random` module

## How It Works

1. A list of paragraphs is created.
2. One paragraph is selected randomly.
3. The selected paragraph is displayed in the terminal.
4. The user presses Enter to start the test.
5. The timer starts.
6. The user types the paragraph.
7. After the user presses Enter, the timer stops.
8. The total time taken is calculated and displayed.
